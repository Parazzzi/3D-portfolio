# Dementiy Besarab 3D Portfolio

Interactive 3D portfolio for Dementiy Besarab, a Unity C# Developer focused on mobile games, VR/XR projects, gameplay systems, UI, integrations, optimization, and project stabilization.

The site keeps the 3D room, monitor, whiteboard, arcade, audio, and puzzle interactions while replacing the personal content with Dementiy's CV, real projects, contact links, certificates, and portfolio assets.

## Local Development

```bash
npm install
npm run dev
```

The dev server will print the local URL, usually `http://localhost:8080`.

## Production Build

```bash
npm run build
```

Webpack writes the production output to `dist/`.

## Main Content Sources

- CV: `static/assets/cv/Dementiy_Besarab_CV.docx`
- PDF CV copy: `static/assets/cv/dementiy-besarab-cv.pdf`
- Project media: `static/assets/portfolio/`
- Screen content: `static/screens/`
